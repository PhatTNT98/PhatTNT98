### 📂 Highlighted Projects

Here are some of my self-prouded projects

#### 1. 🛒 [AHBD-Emarket](https://github.com/PhatTNT98)
*The Online player-based trading market for the specific game - Anh Hùng Bất Diệt*
- **Tech:** SpringBoot, Thymleaf, SQLServer, Postman 

- **Hightlighted functions:**
  - Xác thực và phân quyền người dùng thông qua JWT (JSON Web Token) kèm cơ chế Refresh Token bảo mật.
  - Quản lý sản phẩm (CRUD) tích hợp lọc giá, phân loại, tìm kiếm toàn văn và phân trang.
  - Tích hợp Stripe API để thanh toán trực tuyến, xử lý bất đồng bộ trạng thái đơn hàng qua Stripe Webhook.
  - Tăng cường bảo mật với các lớp lọc mã độc XSS và giới hạn tần suất yêu cầu (Rate Limiting).

#### 2. 💬 [Real-time Chat System](https://github.com/ternal-qtri/realtime-chat-go)
*Kiến trúc nhắn tin thời gian thực hiệu năng cao.*
- **Công nghệ:** Golang, Gorilla WebSocket, Redis Pub/Sub, PostgreSQL, Docker
- **Tính năng nổi bật:**
  - Kết nối hai chiều thời gian thực (duplex communication) qua giao thức WebSocket.
  - Hỗ trợ chat nhóm công khai và nhắn tin riêng trực tiếp (1-1).
  - Đồng bộ hóa tin nhắn đa máy chủ (Horizontal Scaling) qua Redis Pub/Sub (Message Broker).
  - Lưu trạng thái online/offline vào Redis và lưu lịch sử hội thoại lâu dài vào PostgreSQL.

#### 3. 📝 [Task Management Microservices](https://github.com/ternal-qtri/task-management-microservices)
*Hệ thống quản lý công việc và dự án áp dụng kiến trúc Microservices.*
- **Công nghệ:** Java, Spring Boot, Spring Cloud, RabbitMQ, gRPC, MySQL, Netflix Eureka
- **Tính năng nổi bật:**
  - Phân rã thành 3 dịch vụ độc lập: Auth & User Service, Task Service, và Notification Service.
  - Đăng ký dịch vụ động với Eureka Server và định tuyến giao tiếp qua API Gateway.
  - Giao tiếp đồng bộ nội bộ hiệu suất cao thông qua gRPC/Protobuf.
  - Gửi email thông báo bất đồng bộ bằng cách đẩy các sự kiện vào hàng đợi RabbitMQ.

#### 4. ✍️ [Blogging Platform API](https://github.com/ternal-qtri/fastapi-blog-api)
*API dịch vụ viết blog tối ưu hóa hiệu năng bằng cơ chế Caching.*
- **Công nghệ:** Python, FastAPI, SQLAlchemy, PostgreSQL, Redis (Cache), Swagger
- **Tính năng nổi bật:**
  - Xây dựng API hoàn toàn bất đồng bộ (async/await) cho hiệu suất xử lý vượt trội.
  - Caching danh sách bài viết phổ biến trên Redis, giúp giảm thời gian phản hồi xuống dưới 5ms.
  - Tự động vô hiệu hóa cache (cache invalidation) khi bài viết có cập nhật hoặc bình luận mới.

#### 5. ⏰ [Job Queue & Notification Service](https://github.com/ternal-qtri/job-queue-service)
*Dịch vụ nền độc lập chuyên xử lý tác vụ nặng và gửi thông báo.*
- **Công nghệ:** Node.js, BullMQ, Redis, Nodemailer, Twilio API
- **Tính năng nổi bật:**
  - Kiến trúc Worker-Producer tách biệt tác vụ nặng khỏi luồng xử lý chính.
  - Gửi email báo cáo định kỳ tự động bằng BullMQ Scheduler.
  - Tự động thử lại (retry) các tác vụ thất bại với cơ chế giãn cách thời gian (exponential backoff).

#### 6. 🛡️ [API Gateway & Rate Limiter](https://github.com/ternal-qtri/api-gateway-go)
*Cổng kết nối API tập trung hỗ trợ phân phối lưu lượng và giới hạn truy cập.*
- **Công nghệ:** Golang, Redis, Gin Gonic, Rate Limiting
- **Tính năng nổi bật:**
  - Định tuyến yêu cầu (Request Routing) từ Client đến các microservices nội bộ.
  - Giới hạn tần suất truy cập (Rate Limiting) theo IP bằng thuật toán Token Bucket trên bộ nhớ Redis (độ trễ dưới 1ms).
  - Tích hợp CORS bảo mật tập trung và ghi nhận log hệ thống.
